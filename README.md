# 🔐 Password Strength Checker

A web application that checks password strength in real time and helps users create secure passwords.

## 🚀 Features
- Real-time password analysis
- Strength meter (Weak / Medium / Strong)
- Show/Hide password 👁️
- Suggestions for improvement

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Use
1. Open index.html in browser
2. Enter a password
3. View strength and suggestions

## 🎯 Purpose
This project helps users understand how to create strong and secure passwords.
